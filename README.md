## What is this?

A collection of **25 production-ready skills** for AI coding agents like Hermes, OpenCode, and Claude Code.

Each skill is a comprehensive guide with examples, checklists, and step-by-step frameworks.

**No installation required.** Just load and use.

---

## Quick Start

```bash
# Clone the repository
git clone https://github.com/dinoudon/udon-collective-skills.git

# Copy to your AI agent's skills directory
cp -r udon-collective-skills/skills/* ~/.hermes/skills/
```

Then load any skill:
```python
skill_view(name='tdd-iron-law')
```
