## 🚀 Quick Start

### For AI Agent

```bash
# Clone the repository
git clone https://github.com/dinoudon/udon-collective-skills.git

# Copy skills to Hermes
cp -r udon-collective-skills/skills/* ~/.hermes/skills/

# Load a skill
hermes skill view spec-driven-development-enhanced
```

### For OpenCode / Claude Code

```bash
# Clone the repository
git clone https://github.com/dinoudon/udon-collective-skills.git

# Copy skills to your agent's skill directory
cp -r udon-collective-skills/skills/* ~/.opencode/skills/
# or
cp -r udon-collective-skills/skills/* ~/.claude-code/skills/
```

### Usage Example

```python
# In AI Agent
skill_view(name='tdd-iron-law')

# Follow the skill's framework
# 1. Write failing test (RED)
# 2. Make it pass (GREEN)
# 3. Refactor (REFACTOR)
# 4. Verify with mutation testing
```
